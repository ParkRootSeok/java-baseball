# 구현 기능 목록

---

## 📎 Game.class

- `playGame` : 게임 실행 함수
- `restartGame` : 게임 재시작 함수

## 📎 Answer.class

- `createAnswer` : 정답 만들기
- `checkAnswer` : 정답 확인
    - 스트라이크 개수 확인 후 게임 종료

## 📎 Count.class

- `countStrike` : 스트라이크 카운트
    - 스트라이크 개수 반환
- `countBall` : 볼 카운트
    - 볼 개수 반환
- `showCount` : 카운트 출력

## 📎 Convert.class

- `separateNumberToDigit` : 입력된 숫자 분리
    - 입력된 숫자를 각 자릿수로 분리한 리스트 반환
